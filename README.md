# UWM Chatbot Demo Website

This is a **demo chatbot website project** created for educational purposes only. It mimics the style and layout of the University of Wisconsin-Milwaukee's official website for practice and demonstration.

## Purpose
- To showcase frontend and backend development using HTML, CSS, JavaScript, Flask, and Azure.
- Demonstrates chatbot interaction, appointment booking, and basic admin dashboard features.
